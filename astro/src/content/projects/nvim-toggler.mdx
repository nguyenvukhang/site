A neovim plugin that toggles between pairs of words.

Uses cases include flipping a boolean, or when writing CSS and
toggling between opposites like `vertical` and `horizontal`.

This more powerful than vim's built-in `<cword>`, as it can support
toggles between pairs with spaces such as '`- [ ]`' and '`- [x]`'.
