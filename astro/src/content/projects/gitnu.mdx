`git-nu` enumerates the `status` output so users can type numbers
instead of filenames.

CI/CD using [GitHub Actions][gh-actions] which builds, tests, and
deploys automatically to [crates.io][crates-io], where it has 1,700+
downloads.

[gh-actions]: https://github.com/nguyenvukhang/gitnu/actions
[crates-io]: https://crates.io/crates/gitnu
