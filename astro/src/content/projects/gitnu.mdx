Gitnu is a `git` wrapper that enumerates the `status` output so
users can type numbers instead of filenames.

CI/CD using [GitHub Actions][gh-actions] and automatically deploys to
[Crates.io][crates-io]

This project aims to be as light as possible and so it has only one
external dependency.

[gh-actions]: https://github.com/nguyenvukhang/gitnu/actions
[crates-io]: https://crates.io/crates/gitnu
